# Site da HÓTICA

## Sobre o projeto

Site institucional desenvolvido para a HÓTICA com o objetivo de apresentar a empresa, seus produtos e facilitar o contato com clientes.

## Funcionalidades

- Página inicial
- Sobre a empresa
- Catálogo de produtos
- Contato via WhatsApp
- Localização
- Responsividade para celulares

## Tecnologias utilizadas

- HTML
- CSS
- JavaScript
